Openlayers Animated Clusters
----------------------------

Providing animated clustering behaviour to openlayers.

Base on the following existing work:
  http://drupal.org/node/1677522
  http://acuriousanimal.com/code/animatedCluster/
  https://github.com/acanimal/AnimatedCluster/blob/master/AnimatedCluster.js
  http://acuriousanimal.com/blog/2012/08/19/animated-marker-cluster-strategy-for-openlayers/

Install
-------

You must download AnimatedCluster.js to:
  sites/all/libraries/AnimatedCluster/AnimatedCluster.js
