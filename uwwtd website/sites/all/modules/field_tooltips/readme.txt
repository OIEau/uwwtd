INTRODUCTION
------------

Current Maintainers:

 * Devin Carlson <http://drupal.org/user/290182>

Field Tooltips provides a user interface for adding tooltips to fields.

Field Tooltips includes three important features:

 * A simple administration interface for adding tooltips to fields.
 * Support for all field types.
 * Support for HTML tooltips.

REQUIREMENTS
------------

Field Tooltips has two dependencies:

Drupal core modules
 * Field

Contributed modules
 * jQuery Update - Configured to use jQuery 1.7 or higher.

INSTALLATION
------------

To install Field Tooltips:

 * Enable Field Tooltips and all of the modules that it requires.
 * Configure the jQuery Update module to use jQuery 1.7 or higher.

USAGE
-----

Field Tooltips automatically adds tooltip settings to field settings pages.

Administrators or users with the permissions required to edit an entity's
fields will be able to configure tooltip settings by visiting the field's
settings page.

Tooltips can contain HTML when combined with an appropriate input format.
